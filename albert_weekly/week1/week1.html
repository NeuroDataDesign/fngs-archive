Visualizing Clarity

From brains to hydrogels to data

Created by Albert Lee

What is Clarity?

Clarity is an imaging technique that uses hydrogels to make brains transparent allowing for damage-free access to key neuronal structures.

ITKsnap Visualization

Pros

  • Raw data
  • Highest order resolution
  • Annotation capabilities

Cons

  • 2D representation
  • Annoation is irrelevant
  • Difficult to manipulate

Solution

3D plotting

Matplotlib - high threshold

Matplotlib - low threshold

Pros

  • Easy
  • High functionality

Cons

  • Low customizability
  • Poor performance
  • Outdated stylistically

Plotly

Pros

  • Professional and elegant
  • Powerful in scope
  • Simple API

Cons

  • Learning curve
  • Expensive/Restricted Access
  • Limited manipulativity

Javascript

Pros

  • Professional and elegant
  • Extremely powerful
  • Completely manipulatable

Cons

  • Steep learning curve
  • Multiple languages
  • Difficult to debug

Conclusion

3D plotting is a superior method towards visualizing Clarity data. More work needs to be done standardizing the thresholding/sampling, however there is much potential when factoring in opacity metrics.
